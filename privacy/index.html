OHBot Privacy Policy

Last updated: May 14, 2026

OHBot is a Discord bot for searching Once Human builds stored by the Once Human Build Planner.

Information processed

  • Discord interaction data required to respond to slash commands.
  • Command options you submit, such as gun search text and HP type.
  • Server and channel identifiers used to enforce command restrictions.
  • Saved build planner data needed to return matching builds.

Information not collected

OHBot does not collect Discord passwords, payment data, private messages, or unrelated server messages. It only receives interaction payloads that Discord sends to the configured endpoint.

Storage

Build planner data is stored in the project's Cloudflare D1 database. Discord command interactions are processed to answer the request and may appear in provider logs for operational troubleshooting.

Sharing

Data is not sold. The service relies on Discord and Cloudflare, so data required to operate the bot passes through those services under their own policies.

Contact

For questions or removal requests, contact the server owner or project maintainer through the Discord server where OHBot is installed.
