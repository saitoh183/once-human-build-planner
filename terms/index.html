OHBot Terms of Service

Last updated: May 14, 2026

OHBot is a Discord bot for the Once Human Build Planner. It provides slash commands that search saved build data and return matching build details inside Discord.

Use of the bot

  • Use OHBot only for Once Human build lookup and related planner features.
  • Do not abuse, spam, automate, or attempt to disrupt the bot, Discord, Cloudflare, or the build planner service.
  • Commands may be restricted to specific Discord servers or channels.

Data handled

OHBot processes Discord interaction data required to answer slash commands, such as the command name, selected options, server/channel identifiers, and user interaction metadata provided by Discord.

The bot reads saved build planner data from the Once Human Build Planner database. It does not need your Discord password and will never ask for it.

Availability

OHBot is provided as-is. Features may change, break, or be removed without notice. Very dramatic, but standard internet plumbing.

Third-party services

OHBot depends on Discord and Cloudflare services. Their own terms and policies also apply when you use the bot.

Contact

For questions, issues, or removal requests, contact the server owner or project maintainer through the Discord server where OHBot is installed.
